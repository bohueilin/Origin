Why speed matters

Generate → verify → patch → regenerate

Cerebras turns one seed map into many curriculum candidates quickly. Origin turns those candidates into a deterministic eval set and exposes verifier patches for the next loop.

  • Weight-Zone Guard: IF payload_weight > 5kg AND current_zone == 'human-zone' THEN signal(REFUSE)
  • Dock-Edge Hard-Stop: IF distance(robot_pos, loading-edge) < 0.2m THEN trigger(EMERGENCY_STOP)
  • Strict-Payload-Check: if payload_weight > 10 and zone == 'human-zone' then refuse
  • Dock-Edge-Hard-Stop: if distance(robot, loading-edge) < 0.2 then halt_and_refuse
  • Strict-Edge-Fence: if position_inside(loading_edge) then state = REFUSE
