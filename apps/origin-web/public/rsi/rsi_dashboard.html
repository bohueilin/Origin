Origin - Robot-Readiness Gym

A bounded gym where a fast proposer is checked by a deterministic verifier and a ground-truth oracle.

Gemma proposes · Origin verifies · the oracle is the judge — a bounded robot-readiness Gym, not production robot certification.
Honest boundary: room-type classification is structural reading, while the v1 safety policy recovers the oracle from raw geometry. The full 36-feature perfect score is an oracle-recovery upper bound, not the headline. The budget pass is the learned mixed-holdout policy result; the reachability pass is an oracle-recovery upper bound because safe_reach_* is a precomputed flood-fill feature matching the oracle's own reachability check. Cerebras/Gemma mines hard cases; Origin's deterministic oracle labels them. Nothing here certifies a physical robot.
