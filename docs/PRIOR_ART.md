# Prior art & credits

Some Origin capabilities were **inspired by** ideas that surfaced in the public AGI-House agent-identity/security/trust cohort. Everything here is a **clean-room reimplementation** in Origin's own stack and design — **no third-party code was copied**. Ideas, architectures, and methods are not copyrightable; we reimplement them, credit the source, and keep our own deterministic-oracle discipline. Where we ever *vendor* a permissively-licensed library, it is listed in `THIRD_PARTY_NOTICES.md` with its license retained.

| Origin capability | Idea we absorbed | Inspired by (link) | How ours differs |
|---|---|---|---|
| **Cordon** (`apps/passport/.../engine/cordon.ts`) | Taint-track agents exposed to untrusted content; the broker refuses to resolve a secret for a tainted agent (secret never fetched); on compromise, freeze only the poisoned sub-tree and measure blast radius. | CORDON — `github.com/ShreeBohara/cordon-agi-house` (no license) · QuarantineAI — `github.com/tmoula/QuarantineAI` (telemetry-only observer) | Clean-room; wired to Origin's `SecretBroker` chokepoint + the tamper-evident `AuditLogger` hash chain; taint is a property of the agent tracked outside its context. |
| **Tell** (`apps/passport/.../engine/tell.ts` + `activationProbe.ts`) | Three-way gate: declared vs. **measured** vs. action. Predicted-plan → observed-action conformance; a white-box activation probe for the open-weight tier. | Agent Polygraph — `github.com/yavol/code-onion` (MIT) · SecureDelegate — `github.com/rajeev595/AGIHouse_hackathon_2026` (no license) · **TaskTracker** (Microsoft Research, activation-delta method — reimplemented from the paper) | Black-box-first (predicted-plan conformance) on our stack; the white-box probe (`activationProbe.ts`) is a real `ProbeSignal` drop-in — a linear-probe drift score on supplied activation deltas that **abstains at confidence 0 without activations**, so it is never claimed on API-only models. |
| **Crucible** (`apps/origin-web/rlkit/crucible.mjs`) + **IAM gym** (`rlkit/iamGym.mjs`) | A verifiable, config-bound agent credential ("reference check") that voids if model/tools/harness change; before/after lift; expert-authored rubrics; an IAM/access-control gym to certify in. | Diploma.ai — `diploma.ai` (no public repo) · Bad-agents (IAM gym) — `github.com/Dariushuangg/bad-agents` | Issued by Origin's **deterministic oracle** (`env:verify` + RSL) — our moat vs. self-authored rubrics. `iamGym.mjs` is a deterministic, fail-closed, least-privilege access-control environment (allow/deny/escalate; catastrophic = security-critical over-grant); `issueIamReferenceCheck` runs an agent's policy → RSL level → a config-bound credential. Emits a **Sigil** signed receipt into Chronos. |
| **Sigil** (`apps/origin-web/rlkit/sigil.mjs`) | A shareable, browser-signed receipt: a third party verifies an evidence artifact offline without trusting the issuer's server. | TrustReceipt — `github.com/ankitshah009/TrustReceipt` (no license) | Clean-room; **ECDSA P-256 / ES256** via Web Crypto (the same code path in Node ≥18 and the browser); the signature covers rlkit's existing `sha256(canonical(payload))` content-address, so a Sigil reuses the digest the rest of Origin already trusts; optional issuer-thumbprint pinning; honest scope (proves integrity + authenticity, **not** real-world identity binding). |
| **Leak-vs-hold** (`apps/passport/.../secrets/leakVsHold.ts`) | Side-by-side: the same prompt-injection makes a key-in-context agent leak while a broker-handle agent holds. | BrokerAI (leak-vs-hold demo pattern) · `aps-1password-demo` — `github.com/aeoess/aps-1password-demo` (no license) | Clean-room; runs the **real** `MockSecretBroker` + the `redact`/`assertNoSecret` sentinel that guards the whole suite — the "leak" is caught by the exact tracer used everywhere else, not a bespoke check. |

## Naming
Origin's forge/pantheon system: **Origin** (the gym/creation) · **Chronos** (the record) · **Cobra** (red-team) · **Janus** (the gate/identity — formerly "Passport") · **Tell** (measured intent) · **Cordon** (containment) · **Crucible** (certification) · **Sigil** (the signed receipt).
